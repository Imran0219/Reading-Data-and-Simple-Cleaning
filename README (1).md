# Reading-Data-and-Simple-Cleaning
Reading Data and Simple Cleaning
1. Loaded Titanic dataset using pandas.
2. Identified missing values using isnull().sum().
3. Dropped Cabin column due to excessive missing data.
4. Filled Age using median to avoid outliers.
5. Filled Embarked using mode.
6. Removed duplicate rows.
7. Converted Survived column to categorical type.
8. Created Age_Group column for better analysis.
9. Created Family_Size feature.
10. Saved cleaned dataset to CSV.
